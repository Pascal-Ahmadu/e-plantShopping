# Paradise Nursery (E-Plant Shopping)

Welcome to **e-plantShopping**, a comprehensive React-based web application for plant enthusiasts!

## Project Overview

**Paradise Nursery** is a digital platform where green meets serenity. Our mission is to provide a wide range of high-quality plants that enhance your surroundings and contribute to a sustainable lifestyle. From air-purifying plants to aromatic fragrant ones, we cater to every plant lover's needs.

## Features

- **Product Collection**: Explore a diverse range of plants including Air Purifying, Aromatic, Insect Repellent, Medicinal, and Low Maintenance varieties.
- **Shopping Cart**: Easily add plants to your cart and manage your selections.
- **About Us**: Learn more about our mission and team of experts dedicated to plant care.
- **Responsive Design**: A modern, premium interface designed for a seamless user experience.

## Tech Stack

- **Frontend**: React.js with Vite
- **State Management**: Redux Toolkit
- **Styling**: Vanilla CSS

## Getting Started

1. Clone the repository: `git clone https://github.com/Pascal-Ahmadu/e-plantShopping.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
